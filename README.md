## Overview

A PsychoPy-based binocular rivalry (BR) paradigm designed to objectively validate aphantasia through imagery-biased rivalry. Participants view competing stimuli (houses vs. faces) through red-cyan anaglyph glasses while imagining one of the stimuli before each trial. The primary outcome is the **Imagery Priming Index (IPI)**, which quantifies the influence of mental imagery on perceptual dominance.

This paradigm was developed as part of a larger study combining:
- Binocular rivalry (objective imagery measure)
- VVIQ / OSIQ self-report scales
- Visual and verbal DRM false memory paradigms

## Paradigm Design

### Structure
- **1 Baseline block** (rivalry only, no imagery instruction)
- **8 Imagery blocks** (4 house-face pairs × 2 conditions: imagine house / imagine face)
- Each block: Encoding (5s) → Imagery Induction (25s) → Ready (2s) → Rivalry (90s) → Rest (30s)
- Total duration: ~22 minutes

### Stimuli
**Face stimuli** were selected from the KDEF database (Lundqvist et al., 1998):
- 4 neutral-expression photographs (2 female: AF07, AF17; 2 male: AM05, AM25)
- Gender-balanced selection to control for own-gender processing asymmetry

> ⚠️ KDEF images are **not included** in this repository due to licensing restrictions.
> Download from: https://www.kdef.se (free for academic use, registration required)

**House stimuli** were generated using AI image generation tools (4 distinct architectural styles).

### Stimulus Preprocessing
All stimuli were preprocessed to control low-level visual confounds:
- Converted to grayscale
- Resized to 512×512 pixels
- Mean luminance normalized (target: 128/255)
- RMS contrast normalized (target: 40)

### Eye-Color Counterbalancing
Red-cyan color-to-eye assignment was counterbalanced across participants based on participant ID (odd = house/red-left; even = house/cyan-right).

### Response Keys
| Key | Meaning |
|-----|---------|
| `F` | Left stimulus dominant |
| `J` | Right stimulus dominant |
| `SPACE` | Mixed / transitional percept |
| `ESC` | Safe quit (data saved) |

## Primary Dependent Variable

**Imagery Priming Index (IPI)**:

$$IPI = \frac{T_{\text{imagined dominant}}}{T_{\text{total rivalry}}} - 0.50$$

- IPI > 0 → Imagery effect present (typical imager)
- IPI ≈ 0 → No imagery effect (consistent with aphantasia)

Net IPI (baseline-corrected) = imagery block IPI − baseline IPI

## Installation

```bash
git clone https://github.com/TukeOwnz/BR_experiment.git
cd BR_experiment
```

Then:
1. Download KDEF stimuli from https://www.kdef.se
2. Run `normalize_stimuli.py` to preprocess your images
3. Run `BR_main.py` in PsychoPy Coder

### Required folder structure
```
BR_experiment/
├── BR_main.py
├── normalize_stimuli.py
├── stimuli_normalized/     ← your preprocessed grayscale images
│   ├── face_F1_norm.png
│   ├── face_F2_norm.png
│   ├── face_M1_norm.png
│   ├── face_M2_norm.png
```

## Data Output

Each session produces a `.csv` file in `data/` with the following columns:

| Column | Description |
|--------|-------------|
| `participant_id` | Participant identifier |
| `block_number` | Block index (0 = baseline) |
| `block_type` | baseline / house_imagery / face_imagery |
| `pair_number` | Stimulus pair (1–4) |
| `imagine_condition` | house / face / none |
| `house_eye` | Which eye received house (left/right) |
| `face_eye` | Which eye received face (left/right) |
| `percept_onset` | Percept start time (s from rivalry onset) |
| `percept_offset` | Percept end time (s) |
| `percept_duration` | Duration (s) |
| `key_pressed` | F / J / SPACE |
| `dominant_stimulus` | house / face / mixed |
| `trial_time` | Absolute timestamp (ISO format) |

> **Analysis note:** Percept durations < 150ms are transition artifacts and should be excluded prior to analysis.
